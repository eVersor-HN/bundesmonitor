# Design-Prototyp

`index.html` direkt im Browser oeffnen. Der Prototyp zeigt:

- Desktop-Navigation
- chronologischen Feed
- Filterchips
- Status- und Geldkennzeichnung
- Anstehend-/Quellenstatus-Spalte
- responsive Mobilansicht

Er enthaelt fiktive Demonstrationsdaten. Keine Inhalte als reale politische Meldungen uebernehmen.
